1. springbootдјҳзјәзӮ№пјҹ

  • вҲҡеҝ«йҖҹжһ„е»әйЎ№зӣ®пјҢйҖҡиҝҮ Intellij IDEA жҲ–иҖ…е®ҳж–№зҡ„ Spring Initializr е°ұеҸҜд»Ҙеҝ«йҖҹеҲӣе»әж–°йЎ№зӣ®

  • вҲҡиҮӘеҠЁй…ҚзҪ®:жҸҗдҫӣдәҶдёҖзі»еҲ—зҡ„ Starter,еҜ№дё»жөҒжЎҶжһ¶зҡ„ж— й…ҚзҪ®йӣҶжҲҗ

  • вҲҡйЎ№зӣ®зӢ¬з«ӢиҝҗиЎҢпјҢеҶ…еөҢservletе®№еҷЁTomcatгҖҒJetty жҲ–иҖ… Undertow

  • жҸҗдҫӣиҝҗиЎҢж—¶еә”з”Ёзӣ‘жҺ§гҖӮSpring Boot жҸҗдҫӣеҹәдәҺ HTTPгҖҒJMXгҖҒSSH еҜ№иҝҗиЎҢж—¶зҡ„йЎ№зӣ®иҝӣиЎҢзӣ‘жҺ§гҖӮ

  • жһҒеӨ§ең°жҸҗй«ҳдәҶејҖеҸ‘гҖҒйғЁзҪІж•ҲзҺҮ

  • еӣ дёәиҮӘеҠЁй…ҚзҪ® Spring Bean зҡ„еҠҹиғҪпјҢжҲ‘д»¬еҸҜиғҪж— жі•зҹҘйҒ“пјҢе“Әдәӣ Bean иў«иҝӣиЎҢеҲӣе»әдәҶгҖӮиҝҷдёӘж—¶еҖҷпјҢеҰӮжһңжҲ‘д»¬жғіиҰҒиҮӘе®ҡд№үдёҖдәӣ Bean пјҢеҸҜиғҪеӯҳеңЁеҶІзӘҒпјҢжҲ–иҖ…дёҚзҹҘйҒ“е®һйҷ…жіЁе…Ҙзҡ„жғ…еҶөгҖӮ

4. Spring Boot жңүе“ӘеҮ з§ҚиҜ»еҸ–й…ҚзҪ®зҡ„ж–№ејҸпјҹ

  • @ValueжіЁи§ЈпјҢиҜ»еҸ–й…ҚзҪ®еҲ°еұһжҖ§гҖӮж”ҜжҢҒе’Ң @PropertySourceжіЁи§ЈдёҖиө·дҪҝз”ЁпјҢжҢҮе®ҡдҪҝз”Ёзҡ„й…ҚзҪ®ж–Үд»¶гҖӮ
  • @ConfigurationPropertiesжіЁи§ЈпјҢиҜ»еҸ–й…ҚзҪ®еҲ°зұ»дёҠгҖӮж”ҜжҢҒе’Ң@PropertySourceжіЁи§ЈдёҖиө·дҪҝз”ЁпјҢжҢҮе®ҡдҪҝз”Ёзҡ„й…ҚзҪ®ж–Үд»¶гҖӮ

5. bootstrap.propertiesе’Ңapplication.propertiesжңүдҪ•еҢәеҲ«?

  • bootstrap(.yml/.properties)пјҡз”ұзҲ¶ApplicationContextеҠ иҪҪпјҢжҜ”applicatonдјҳе…ҲеҠ иҪҪпјҢй…ҚзҪ®еңЁеә”з”ЁзЁӢеәҸдёҠдёӢж–Үзҡ„еј•еҜјйҳ¶ж®өз”ҹж•ҲгҖӮеңЁSpringCloud ConfigжҲ–иҖ…Nacosдёӯдјҡз”ЁеҲ°гҖӮдё”еұһжҖ§дёҚиғҪиў«иҰҶзӣ–пјӣ
  • application(.yml/.properties)пјҡз”ұApplicatonContextеҠ иҪҪпјҢз”ЁдәҺйЎ№зӣ®зҡ„иҮӘеҠЁеҢ–й…ҚзҪ®гҖӮ

public class CorsConfig implements WebMvcConfigurer {

    public void addCorsMappings(CorsRegistry registry) {
        registry.addMapping("/**")

7. springbootй…ҚзҪ®ж–Үд»¶иҜ»еҸ–йЎәеәҸпјҹвҲҡ

  • config/application.properties|ymlпјҲйЎ№зӣ®ж №зӣ®еҪ•дёӯconfigзӣ®еҪ•дёӢпјү

  • application.properties|ymlпјҲйЎ№зӣ®ж №зӣ®еҪ•дёӢпјү

  • resources/config/application.properties|ymlпјҲйЎ№зӣ®resourcesзӣ®еҪ•дёӯconfigзӣ®еҪ•дёӢпјү

  • resources/application.properties|ymlпјҲйЎ№зӣ®зҡ„resourcesзӣ®еҪ•дёӢпјү

  • еҰӮжһңеҗҢдёҖдёӘзӣ®еҪ•дёӢпјҢй»ҳи®Өе…ҲиҜ»еҸ–application.propertiesгҖӮз„¶еҗҺapplication.yml

  • еҰӮжһңеҗҢдёҖдёӘй…ҚзҪ®еұһжҖ§пјҢй»ҳи®ӨдҪҝз”Ёз¬¬1дёӘиҜ»еҸ–еҲ°зҡ„

  • й…ҚзҪ®ж–Үд»¶ж”ҫзҪ®еңЁйЎ№зӣ®зҡ„resourcesзӣ®еҪ•дёӢпјҢеӣ дёәй…ҚзҪ®ж–Үд»¶зҡ„дҝ®ж”№пјҢйҖҡиҝҮзғӯйғЁзҪІдёҚз”ЁйҮҚж–°еҗҜеҠЁйЎ№зӣ®пјҢзғӯйғЁзҪІдҪңз”ЁиҢғеӣҙжҳҜclasspathдёӢ

8. springbootе®һзҺ°е°Ҷй…ҚзҪ®ж–Үд»¶ж”ҫеҲ°classpathпјҢйӮЈд№ҲжүҖжңүзҡ„ж–Үд»¶е°ҶдёҚеҸ—жүҖж”ҫдҪҚзҪ®зҡ„еҪұе“Қ

  • Xbootclasspath/a:дёҺи·Ҝеҫ„ж— з©әж ј и·Ҝеҫ„дёәеҗҢjarи·Ҝеҫ„дёӢзҡ„configж–Үд»¶пјҢжӯӨж—¶жүҖжңүзҡ„й…ҚзҪ®ж–Үд»¶йғҪдјҡеғҸclasspathдёҖж ·еҠ иҪҪ
  • java -Xbootclasspath/a:/config/ -jar testspringbootMybaits-0.0.1-SNAPSHOT.jarжіЁж„ҸжӯӨж—¶зҡ„й…ҚзҪ®ж–Үд»¶дјҡиҰҶзӣ–еҺҹжқҘзҡ„й…ҚзҪ®пјҒпјҒпјҒпјҒпјҒпјҒ
  • /config/ж №зӣ®еҪ•пјҢconfig/зӣёеҜ№еҪ“еүҚзӣ®еҪ•
  • jarеҢ…жҢҮе®ҡзј–з ҒеҗҜеҠЁ java -Dfile.encoding=utf-8 -jar test.jar

9. SpringBoot иҮӘеҠЁй…ҚзҪ®еҺҹзҗҶгҖҒ@EnableAutoConfigurationе®һзҺ°еҺҹзҗҶ

  • @SpringBootApplicationжіЁи§ЈеҢ…еҗ«дәҶ@EnableAutoConfigurationпјҢ@ConfigurationпјҲжҢҮе®ҡзұ»жҳҜ Bean е®ҡд№үзҡ„й…ҚзҪ®зұ»пјүпјҢ@ComponentScan(жү«жҸҸжҢҮе®ҡеҢ…дёӢзҡ„ Bean),иҖҢ@EnableAutoConfiguration(жү“ејҖиҮӘеҠЁй…ҚзҪ®еҠҹиғҪ)дҪҝз”ЁдәҶ@Import({AutoConfigurationImportSelector.class}) иҖҢAutoConfigurationImportSelectorпјҢдҪҝз”ЁImportSelectorжҺҘеҸЈзҡ„selectImportsж–№жі•еҠ иҪҪMETA-INF/spring.factoriesж–Үд»¶зҡ„дҝЎжҒҜorg.springframework.boot.autoconfigure.EnableAutoConfiguration=пјҢе°ҒиЈ…жҲҗbeanNameпјҢеҠ иҪҪеҲ°IOCе®№еҷЁдёӯпјҢе®һзҺ°иҮӘеҠЁй…ҚзҪ®еҠҹиғҪ
  • SpringBootеңЁеҗҜеҠЁж—¶жү«жҸҸйЎ№зӣ®жүҖдҫқиө–зҡ„jarеҢ…пјҢеҜ»жүҫеҢ…еҗ«spring.factoriesж–Үд»¶зҡ„jar
  • ж №жҚ®spring.factoriesй…ҚзҪ®еҠ иҪҪAutoConfigureзұ»
  • ж №жҚ®[@Conditional зӯүжқЎд»¶жіЁи§Ј](Spring Boot жқЎд»¶жіЁи§Ј) зҡ„жқЎд»¶пјҢиҝӣиЎҢиҮӘеҠЁй…ҚзҪ®е№¶е°ҶBeanжіЁе…ҘSpring IoCдёӯ

Springboot еҗҜеҠЁеҺҹзҗҶпјҹ

  • SpringApplicationиҝҷдёӘзұ»дё»иҰҒеҒҡдәҶд»ҘдёӢеӣӣд»¶дәӢжғ…пјҡ
    • жҺЁж–ӯеә”з”Ёзҡ„зұ»еһӢжҳҜжҷ®йҖҡзҡ„йЎ№зӣ®иҝҳжҳҜ Web йЎ№зӣ®
    • жҹҘжүҫе№¶еҠ иҪҪжүҖжңүеҸҜз”ЁеҲқе§ӢеҢ–еҷЁ пјҢ и®ҫзҪ®еҲ° initializers еұһжҖ§дёӯ
    • жүҫеҮәжүҖжңүзҡ„еә”з”ЁзЁӢеәҸзӣ‘еҗ¬еҷЁпјҢи®ҫзҪ®еҲ° listeners еұһжҖ§дёӯ
    • жҺЁж–ӯе№¶и®ҫзҪ® main ж–№жі•зҡ„е®ҡд№үзұ»пјҢжүҫеҲ°иҝҗиЎҢзҡ„дё»зұ»
  • дёәд»Җд№ҲSpringBootеңЁеҗҜеҠЁзҡ„ж—¶еҖҷиғҪеӨҹжүҫеҲ°mainж–№жі•дёҠзҡ„@SpringBootApplicationжіЁи§Јпјҹ
  • еӣ дёәе®ғеҲ©з”ЁдәҶJavaзҡ„еҸҚе°„жңәеҲ¶е’Ңзұ»еҠ иҪҪжңәеҲ¶пјҢSpringApplication.run(Class<?>primarySource,String...args)ж–№жі•жҺҘж”¶дёӨдёӘеҸӮж•°пјҡз¬¬дёҖдёӘжҳҜдё»еә”з”Ёзұ»пјҲеҚіеҢ…еҗ«mainж–№жі•зҡ„зұ»пјүпјҢз¬¬дәҢдёӘжҳҜе‘Ҫд»ӨиЎҢеҸӮж•°гҖӮprimarySourceеҸӮж•°жҸҗдҫӣдәҶдёҖдёӘиө·зӮ№пјҢSpringBootйҖҡиҝҮе®ғжқҘеҠ иҪҪеә”з”ЁдёҠдёӢж–ҮгҖӮ
  • SpringBootеҲ©з”ЁJavaеҸҚе°„жңәеҲ¶жқҘиҜ»еҸ–дј йҖ’з»ҷrunж–№жі•зҡ„зұ»пјҲMyApplication.classпјүгҖӮе®ғдјҡжЈҖжҹҘиҝҷдёӘзұ»дёҠзҡ„жіЁи§ЈпјҢеҢ…жӢ¬@SpringBootApplicationгҖӮ
