java常用api

api：Application(应用) Programming(程序) Interface(接口)

  • java API就是jdk中提供给我们使用的类，这些类将底层的代码实现封装了起来，
  • 不需要关心这些类是如何实现的，只需要学习这些类如何使用即可。
  • 源码：JDK安装目录的src.zip

字符串是常量，在创建之后不能更改，内存区域中就生成了这个字符串。
但引用字符串的变量中记录的地址值是可以改变的
字符串的值本身不能改变，一旦没有引用就会被垃圾回收机制回收
