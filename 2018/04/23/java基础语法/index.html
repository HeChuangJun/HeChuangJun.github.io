Java开发环境的搭建

Java概述

Java是一门编程语言是用来编写软件的，主要有以下两种
    互联网：电商、P2P等等；企业级应用：ERP、CRM、BOS、OA

7个常用dos命令(win+R+cmd:编译和运行时java程序使用)

清屏：cls
    切换盘符：例如：C:\ >E:，回车就到了E盘下。一定要回到盘符根目录使用
    进入指定目录：cd；例如D:\>cd develop\Java，这样就进入了D:\develop\Java目录
    返回上一级目录：cd..；0 例如：D:\develop\Java>cd..，回车，就返回到D:\develop目录下
    返回盘符根目录：cd\；
    显示当前目录下的文件和子目录信息：dir；
    运行记事本程序：D:\>  C:\windows\notepad[.exe]

Java开发环境概述（Java程序运行需要的环境）

JDK：Java Development Kit，程序员的电脑上安装JDK；JDK包含JRE
    JRE：Java Runtime Environment，Java运行环境，用户在电脑上安装JRE即可

JDK下载&安装&校验（WIN+R+CMD+JDK安装路径\bin\java.exe）

Java跨平台（操作系统 （Windows，Linux，Mac））特性

Java程序可以在任意操作系统上运行，一次编写到处运行
    实现跨平台需要依赖Java的虚拟机JVM（Java Virtual Machine）

HelloWorld案例

编写Java源文件（源代码文件），扩展名为.java

使用最普通的文本编辑器编写，修改文件名称为HelloWorld.java
    注意文件名称的大小写。以及是否已经修改了文件的扩展名
    类名必须与源文件名称相同
    public class HelloWorld {
        public static void main(String[] args) {
            System.out.println("HelloWorld");
        }
    }

编译：通过编译器把源文件编译成字节码文件，扩展名为.class

编译器：JDK目录下的bin目录javac.exe
    编译命令为：D:\develop\Java\jdk1.7.0_72\bin\javac.exe D:\java\HelloWorld.java
    编译之后会在源文件同目录下生成字节码文件，字节码文件的扩展名为.class
    编译时控制台上不会有输出，如果有输出说明源代码有错误

运行：使用解释器来运行字节码文件

D:\develop\Java\jdk1.7.0_72\bin\java.exe HelloWord；
    在运行HelloWorld.class文件时不能给出“.class”，切记！！！

配置环境变量（减少输入JDK的bin目录的麻烦）

在DOS控制台中使用的程序只给出程序名称，而没有给出完整路径时，
    Windows系统会到Path变量保存的路径中查找程序
    鼠标右键点击计算机→属性→高级系统设置→环境变量→系统变量中的Path→编辑
    将JDK安装目录\bin路径配置到PATH变量中，用英文分号与其他变量隔开
    编译：D:/java/>javac HelloWorld.java；
    运行：D:/java/>java HelloWorld；

注释、关键字与标识符

程序注释：对源码进行解释说明的文字

单行注释//注释内容
    多行注释/*注释内容*/
    文档注释/**注释内容*/

关键字：被赋予特殊含义，具有专门用途的均为小写的单词

标识符

类名、方法名、变量名、包名、接口名
    组成元素：英文字符: a-zA-Z、数字: 0-9、符号: _与$
    规则：数字不能开头、不可以使用关键字、严格区分大小写，见名知意
    命名规范
        类名&接口名：大驼峰式AaaBbbCcc  
        变量名&方法名：小驼峰式aaaBbbCcc
        常量名：多单词组成是所有字母均大写，使用_连接AAA_BBB_CCC
        包名：多单词组成时所有字母均小写，使用.连接aaa.bbb.ccc

数据类型：基本数据类型和引用数据类型（数组、类、接口）

基本数据类型

四类 八种 字节数 数据表示范围
整型 byte 1 -128～127 7次方
short 2 -32768～32767 15次方
int 4 -2147483648～2147483648 31次方
long 8 -263～263-1 63次方+L
浮点型 float 4 -3.403E38～3.403E38 128次方～127次方+F 7位有效数字
double 8 -1.798E308～1.798E308 1024次方～1023次方+D 16位~
字符型 char 2 ‘’包裹，只包含一字符，运算按ASCII码对应的整数运算
布尔型 boolean 1 true、false

不同数据类型的初始值

数据类型 默认初始化值
byte、short、int、long 0
float、double 0.0
char 一个空字符（空格），即’\u00000’
boolean false
引用数据类型 null，表示变量不引用任何对象

数据类型转换（布尔值除外）

不同类型的变量可以在一起运算，但要先进行类型转换再运算
    范围小的数据类型值（如byte），可以直接转换为范围大的数据类型值（如int）
    范围大的数据类型值（如int），不可以直接转换为范围小的数据类型值（如byte）
    数据类型数据范围从小到大：byte -> short -> int -> long -> float -> double
    自动类型转换：范围小的数据类型转换成范围大的数据类型
    格式：范围大的数据类型 变量 = 范围小的数据类型值；
    强制类型转换：范围大的数据类型转换成范围小的数据类型
    格式：范围小的数据类型  变量 = (范围小的数据类型) 范围大的数据类型值;
    多种类型数据混合运算时，系统自动将所有数据转换成容量最大的那一种数据类型再计算。

常量：不变的数据量

数据类型 举例（字符均指汉字数字和字母）
整数类型 二进制：以0零b(0B)开头 如0b1011 、0B1001
八进制：以0开头 如01、07、0721
十进制：正常数字 如 13、25等
十六进制：以0x(0X)开头,数字以0-9及A-F组成,如0x23A2、0xa、0x10
浮点类型 1.0、-3.15、3.168等
字符类型 字符必须使用’’ 包裹，并且其中只能且仅能包含一个字符
布尔类型 true、false
字符串类型(引用类型) 字符串必须使用””包裹，可以包含0~N个字符

进制转换

其他进制到十进制：把系数*基数的权次幂相加即可。
    十进制到其他进制转换：除基取余，直到商为0，最后，余数反转就是结果

原码、补码、反码

原码：二进制数的有符号表示法，即最高位为符号位，
    反码：A：正数 与原码相同B：负数 符号位不变，数值位按位取反，0变1,1变0
    补码：A：正数 与原码相同B：负数 反码加1。

常量的类型

整数常量可以根据所在范围来确定类型、默认的整数类型是int类型

不在int范围内的常量必须添加“L”后缀（小写的也可以，建议大写）
      在int范围内的常量也可以添加“L”后缀。添加了“L”后缀的整数常量都是long类型

默认的浮点类型是double类型

所有没有后缀以及使用“D”后缀的小数都是double类型；
     float类型常量必须添加“F”后缀，

变量

内存中装载数据的小盒子，用来存数据和取数据
    变量必须要有明确的类型，同时也要指定变量能装载多大的数据
    定义变量：数据类型  变量名  =  数据值；
    变量名是合法的标识符。在程序中可通过变量名来操作变量。给变量赋的值要与类型符合

    变量定义后可以不赋值，使用时再赋值。不赋值不能使用
    变量不可以重复定义
    变量使用时有作用域的限制

运算符

运算符是用来计算数据的符号。数据为常量或变量。被运算符操作的数我们称为操作数。

算术运算符:+、+、+、-、-、*、/、%、++、–

加法运算符在连接字符串时只有直接与字符串相加才会转成字符串
    “%”小数取余没有意义。结果符号与被取余符号相同
    “/”当两边为整数时，取整数部分，舍余数
    当其中一边为浮点型时，按正常规则相除。整数做被除数，0不能做除数
    ++,--运算符单独使用，不参与运算操作时，运算符前后位置导致的运算结果一致
    ++,--运算符后置时，变量a先参与运算操作，然后变量a自增1或者自减1
    ++,--运算符前置时，变量a的值先自增1或者自减1，然后参与运算操作

赋值运算符:=、+=、-=、*=、/=、%=

将结果自动强转成等号左边的数据类型,左边必须是变量

比较运算符:==、！=、<、>、<=、>=

返回布尔值，判断两个操作数的大小关系及是否相等关系
    赋值运算符的=符号，是用来将=符号右边的值，赋值给=符号左边的变量
    ==符号对于基本数字类型的变量来说是用来判断左右变量的值是否相等的
    对于对象来说，比较的是内存地址

逻辑运算符:&（与）、|、^、！、&&、||

连接两个其他表达式计算后的布尔值结果
    当使用短路与或者短路或时，只要能判断出结果则后边的部分就不再判断。
    异或^： 左右两边条件结果相同，结果就为false，左右两边条件结果不同，结果就为true；

三元运算符:(条件表达式)？表达式1：表达式2；

先判断条件表达式的值，若为true，运算结果为表达式1；若为false，运算结果为表达式2
    System.out.println( 3>2 ? “正确” : “错误” ); 
    int a = 3;int b = 4;String result = (a==b) ? “相等” : “不相等”;  
    int n = (3>2 && 4>6) ? 100 : 200;

位运算符

&：仅当两个操作数都为1时。输出结果才为1。否则为0
    |：仅当两个操作数都为0时，输出的结果才为0。
    ^：仅当两个操作数不同一时候。对应的输出结果才为1，否则为0
    ~：取反'运算符~的作用是将各位数字取反：全部的0置为1，1置为0
    <<：左移就是把一个数的全部位数都向左移动若干位
    >>：右移就是把一个数的全部位数都向右移动若干位
    >>>：无符号右移一位。不足的补0

优先级 描述 运算符 优先级 描述 运算符

流程控制

顺序结构

选择结构

①选择结构if

if语句判断条件是一个布尔值，当判断条件为true时，{}中的执行语句才会执行
        if (条件语句){ 
            执行语句;
            ……
        }
    if…else语句指如果判断条件为true时，就进行某种处理，否则就进行另一种处理
        if (判断条件){
            执行语句1
            ……
        }else{
            执行语句2
            ……
        }//与三元运算符功能相似
    if…else if…else语句所有条件均未满足，else后面{}中的执行语句n+1会执行
        if (判断条件1) {
            执行语句1
        } else if (判断条件2) {
            执行语句2
        }
        ...
        else if (判断条件n) {
            执行语句n
        } else {
            执行语句n+1
        }

②选择结构switch

if…else if…else语句来实现，但是由于判断条件比较多，实现起来代码过长，不便于阅读。
        switch (表达式){
        case 目标值1:
            执行语句1
            break;
        case 目标值2:
        case目标值3:
            执行语句2
            break;
        ．．．．．．
        case 目标值n:
            执行语句n
            break;
        default:
            执行语句n+1
            break;
        }
    表达式的值为byte、short、int、char、enum枚举、String

循环结构

循环语句while
        反复进行条件判断，条件成立，{}内的执行语句就会执行，直到条件不成立，while循环结束
        while(循环条件表达式){//循环体
        执行语句...
        控制条件表达式;
        }
    循环语句do…while
        do {
        执行语句………
        控制条件表达式;
        } while(循环条件表达式);
        循环体会无条件执行一次，然后再根据循环条件来决定是否继续执行
    循环语句for
        一般用在循环次数已知的情况下
        for（初始化表达式; 循环条件; 操作表达式）{//1243243
            执行语句
            ………
        }
    无限循环:while(true){} 或for(;;){}
        无限循环存在的原因是并不知道循环多少次，而是根据某些条件，来控制循环
    循环嵌套:在一个循环语句的循环体中再定义一个循环语句的语法结构

跳转语句

break语句

在switch条件语句中时，终止某个case并跳出switch结构
    在循环语句中，作跳出本层循环语句，执行后面的代码
    标记：当break语句出现在嵌套循环中的内层循环时，它只能跳出内层循环，
    如果想使用break语句跳出外层循环则需要对外层循环添加   标记XXX：然后break XXX

continue语句

终止本次循环，执行下一次循环
    continue语句后面也可以通过使用标记的方式结束本次外层循环，用法与break语句相似
    public class BreakDemo {
    public static void main(String[] args) {
    int i, j; // 定义两个循环变量
    itcast: for (i = 1; i <= 9; i++) { // 外层循环
    for (j = 1; j <= i; j++) { // 内层循环
    if (i > 4) { // 判断i的值是否大于4
    break itcast; // 跳出外层循环
    }
    System.out.print("*"); // 打印*
    }
    System.out.print("\n"); // 换行
    }
    }
    }//只打印了4个*

数组

一维数组

数组是用来存储同一个数据类型多个元素的一个容器
    （数组长度是固定的，数组中存储的元素的数据类型要求一致）

数组定义：

①动态初始化（定义数组时只指定数组长度）
数据类型[] 数组名 = new 数据类型[元素个数或数组长度];
    int[] x;// 声明一个int[]类型的变量
    x = new int[100];// 创建一个长度为100的数组
    int[] x; 声明了一个变量x，该变量的类型为int[]，即一个int类型的数组。
    变量x会占用一块内存单元，它没有被分配初始值。
    x = new int[100]; 创建了一个数组，将数组的地址赋值给变量x。
    在程序运行期间可以使用变量x来引用数组，这时内存中的状态会发生变化
②静态初始化（定义数组时为元素赋值）
类型[] 数组名 = new 类型[]{元素，元素，……};
    类型[] 数组名 = {元素，元素，元素，……};（建议）
访问
数组名[索引/角标]   索引从0开始到数组长度-1
    数组名[索引/角标] =值
    数组长度：数组名.length
遍历
for (int i = 0; i < arr.length; i++) {
        System.out.println(arr[i]); // 通过索引访问元素
    }
数组最值
int[] arr = { 4, 1, 6, 3, 9, 8 }; 
    int max = arr[0];
    for (int x = 1; x < arr.length; x++){
        if (arr[x] > max) { 
            max = arr[x];
        }
    }

二维数组:包含多个一维数组的数组，数组中每个元素都是一维数组

定义

int[][] arr = new int[m][n];
    m: 代表二维数组中一维数组的个数,n: 代表每个一维数组中元素的个数
    int[][] arr = new int[3][];
    m: 代表二维数组中一维数组的个数,每一个一维数组通过赋值来确定数组长度
    int[][] arr = {{1,2},{3,4,5,6},{7,8,9}};

访问

数组名[索引][索引]
    数组名[索引][索引]=值
    数组名.length

遍历

for (int i=0; i<arr2.length; i++) {
    for (int j=0; j<arr2[i].length; j++) {
        sum2 += arr2[i][j];
    }
    }

空指针异常NullPointerException：变量引用数组时，变量的值必须是数组对象，不能为null
    数组越界异常ArrayIndexOutOfBoundsException
