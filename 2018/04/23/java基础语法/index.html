    java开发环境的搭建

    java概述

    java是用来编写互联网(电商、P2P)和企业级应用(ERP、CRM、BOS、OA)等软件的一门编程语言
java跨平台（操作系统 （Windows，Linux，Mac））特性
java程序可以在任意操作系统上运行，一次编写到处运行
实现跨平台需要依赖java的虚拟机JVM（java Virtual Machine）

    java开发环境概述（java程序运行需要的环境）

    JDK：java Development Kit，程序员的电脑上安装JDK；JDK包含JRE
JRE：java Runtime Environment，java运行环境，用户在电脑上安装JRE即可

    HelloWorld程序

    ①编写java源代码文件
用记事本新建文件并命名为HelloWorld.java
类名必须与源文件名称完全相同(扩展名为.java)
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("HelloWorld");
    }
}

②编译：通过编译器把源文件编译成字节码文件(扩展名为.class)
编译器：JDK目录下的bin目录javac.exe
编译命令为：D:\develop\java\jdk1.7.0_72\bin\javac.exe D:\java\HelloWorld.java
编译之后会在源文件同目录下生成字节码文件，字节码文件的扩展名为.class
编译时控制台上不会有输出，如果有输出说明源代码有错误

③运行：使用解释器来运行字节码文件
D:\develop\java\jdk1.7.0_72\bin\java.exe HelloWord
在运行HelloWorld.class文件时不能给出“.class”，切记！！！

    配置java环境变量（减少输入JDK的bin目录的麻烦）

    在DOS控制台中使用的程序只给出程序名称，而没有给出完整路径时，Windows系统会到Path变量保存的路径中查找程序
鼠标右键点击计算机→属性→高级系统设置→环境变量→系统变量中的Path→编辑，将JDK安装目录\bin路径配置到PATH变量中，用英文分号与其他变量隔开
编译：D:/java/>javac HelloWorld.java；
运行：D:/java/>java HelloWorld；

    注释、关键字与标识符

    类别 说明 举例
    程序注释 对源码进行解释说明的文字 //单行注释、/*多行注释*/、/**文档注释**/
    关键字 被赋予特殊含义，具有专门用途的均为小写的单词 public等
    标识符 类、方法、变量、包、接口名 组成元素 a-zA-Z、0-9、_与$
    规则 数字不能开头、不可以使用关键字、严格区分大小写，见名知意
    命名规范 类名&接口名：大驼峰式AaaBbbCcc
    变量名&方法名：小驼峰式aaaBbbCcc
    常量名：多单词组成是所有字母均大写，使用_连接AAA_BBB_CCC
    包名：多单词组成时所有字母均小写，使用.连接aaa.bbb.ccc

    数据类型分类：基本数据类型和引用数据类型

    基本数据类型 八种 字节数 数据表示范围 默认值 包装类
    整型 byte 1 -2^7~2^7-1 0 Byte
    short 2 -2^15～2^15 0 Short
    int 4 -2^31～2^31-1 31次方 0 Integer
    long 8 -2^63～2^63-1 63次方+L 0 Long
    浮点型 float 4 -3.403E38～3.403E38 128次方～127次方+F 7位有效数字 0.0 Float
    double 8 -1.798E308～1.798E308 1024次方～1023次方+D 16位~ 0.0 Double
    字符型 char 2 ‘’包裹，只包含一字符，运算按ASCII码对应的整数运算 ‘’ Character
    布尔型 boolean 1 true、false false Boolean
    引用数据类型 数组、类、接口 null，表示变量不引用任何对象 
    数据类型转换（布尔值除外）
不同类型的变量可以在一起运算，但要先进行类型转换再运算
范围小的数据类型值（如byte），可以直接转换为范围大的数据类型值（如int）
范围大的数据类型值（如int），不可以直接转换为范围小的数据类型值（如byte）
数据类型数据范围从小到大：byte -> short -> int -> long -> float -> double
自动类型转换：范围小的数据类型转换成范围大的数据类型
格式：范围大的数据类型 变量 = 范围小的数据类型值；
强制类型转换：范围大的数据类型转换成范围小的数据类型
格式：范围小的数据类型  变量 = (范围小的数据类型) 范围大的数据类型值;
多种类型数据混合运算时，系统自动将所有数据转换成容量最大的那一种数据类型再计算

    常量

    常量(不变的数据量)数据类型 举例（字符均指汉字数字和字母）
    整数类型 二进制：以0零b(0B)开头 如0b1011 、0B1001
    八进制：以0开头 如01、07、0721
    十进制：正常数字 如 13、25等
    十六进制：以0x(0X)开头,数字以0-9及A-F组成,如0x23A2、0xa、0x10
    其他进制到十进制：把系数*基数的权次幂相加即可。
    十进制到其他进制转换：除基取余，直到商为0，最后，余数反转就是结果
    整数常量根据所在范围来确定类型、默认的整数类型是int类型
    浮点类型 1.0、-3.15、3.168等
    不在int范围内的常量必须添加“L”后缀（小写的也可以，建议大写）
    在int范围内的常量也可以添加“L”后缀。添加了“L”后缀的整数常量都是long类型
    所有无后缀以及使用“D”后缀的小数都是double类型
    float类型常量必须添加“F”后缀，默认的浮点类型是double类型
    字符类型 字符必须使用’’ 包裹，并且其中只能且仅能包含一个字符
    布尔类型 true、false
    字符串类型String(引用类型) 字符串必须使用””包裹，可以包含0~N个字符 
    原码：二进制数的有符号表示法，即最高位为符号位，
反码：A：正数 与原码相同B：负数 符号位不变，数值位按位取反，0变1,1变0
补码：A：正数 与原码相同B：负数 反码加1。

    变量

    变量:内存中装载数据的小盒子，用来存数据和取数据
定义变量：数据类型  变量名  =  数据值；

变量定义后可以不赋值，使用时再赋值。不赋值不能使用
变量不可以重复定义
变量使用时有作用域的限制

    运算符

    运算符是用来计算数据的符号。数据为常量或变量。被运算符操作的数我们称为操作数。
    运算符 举例 说明
    算术运算符 -(减、负号)、*、+(正、加号、连接字符串) 加法运算符在连接字符串时只有直接与字符串相加才会转成字符串
    %(取模) 小数取余没有意义。结果符号与被取余符号相同
    / 均为整数则取整舍余，当其中一边为浮点型则按正常规则相除。除数不为0
    ++(自增)–(自减) 单独使用，不参与运算操作时，运算符前后位置导致的运算结果一致
    后置时运算后才自增/减1，前置则运算前自增/减1
    赋值运算符 +=、-=、*=、/=、%= 将结果自动强转成等号左边的数据类型,左边必须是变量
    = 将=符号右边的值，赋值给=符号左边的变量
    比较运算符 !=、<、>、<=、>= 返回布尔值，判断两个操作数的大小关系及是否相等关系
    == 比较左右基本数字类型的变量值或者对象的内存地址是否相等
    逻辑运算符 &(与)、¦(或)、 连接两个其他表达式计算后的布尔值结果
    &&(短路与)、¦¦(短路或) 只要能判断出结果则后边的部分就不再判断
    逻辑非! 将true变false，false变true
    异或^： 左右两边条件结果相同，结果就为false，否则true；
    三元运算符 (条件表达式)？表达式1：表达式2； 条件表达式的值若为true，结果为表达式1；否则为表达式2
    System.out.println( 3>2 ? “正确” : “错误” );
    int a = 3;int b = 4;String result = (a==b) ? “相等” : “不相等”;
    int n = (3>2 && 4>6) ? 100 : 200;
    位运算符 &(按位与) 仅当两个操作数都为1时。输出结果才为1。否则为0
    ¦(按位或) 仅当两个操作数都为0时，输出的结果才为0。
    ^(异或) 仅当两个操作数不同一时候。对应的输出结果才为1，否则为0
    ~(取反) 将各位数字取反：全部的0置为1，1置为0
    << 左移就是把一个数的全部位数都向左移动若干位
    >> 右移就是把一个数的全部位数都向右移动若干位
    >>> 无符号右移一位。不足的补0


    运算符优先级 描述 运算符 优先级 描述 运算符
    1 括号 ()、[] 9 按位与 &
    2 正负号 +、- 10 按位异或 ^
    3 自增自减，非 ++、–、! 11 按位或 |
    4 乘除，取余 *、/、% 12 逻辑与 &&
    5 加减 +、- 13 逻辑或 ||
    6 移位运算 <<、>>、>>> 14 条件运算 ?:
    7 大小关系 >、>=、<、<= 15 赋值运算 =、+=、-=、*=、/=、%=
    8 相等关系 ==、!= 16 位赋值运算 &=、|=、<<=、>>=、>>>=

    选择结构

    ①选择结构if
if语句判断条件是一个布尔值，当判断条件为true时，{}中的执行语句才会执行
    if (条件语句){ 
        执行语句;
        ……
    }
if…else语句指如果判断条件为true时，就进行某种处理，否则就进行另一种处理
    if (判断条件){
        执行语句1
        ……
    }else{
        执行语句2
        ……
    }//与三元运算符功能相似
if…else if…else语句所有条件均未满足，else后面{}中的执行语句n+1会执行
    if (判断条件1) {
        执行语句1
    } else if (判断条件2) {
        执行语句2
    }
    ...
    else if (判断条件n) {
        执行语句n
    } else {
        执行语句n+1
    }


②选择结构switch
if…else if…else语句来实现，但是由于判断条件比较多，实现起来代码过长，不便于阅读。
    switch (表达式){//表达式的值为byte、short、int、char、enum枚举、String
    case 目标值1:
        执行语句1
        break;
    case 目标值2:
    case 目标值3:
        执行语句2//多个条件满足
        break;
    ．．．．．．
    case 目标值n:
        执行语句n
        break;
    default:
        执行语句n+1
        break;
    }

    循环结构

    循环语句while
    反复进行条件判断，条件成立，{}内的执行语句就会执行，直到条件不成立，while循环结束
    while(循环条件表达式){//循环体
        执行语句...
        控制条件表达式;
    }
循环语句do…while
    循环体会无条件执行一次，然后再根据循环条件来决定是否继续执行
    do {
        执行语句………
        控制条件表达式;
    } while(循环条件表达式);

循环语句for
    一般用在循环次数已知的情况下
    for（初始化表达式; 循环条件; 操作表达式）{//1243243
        执行语句
        ………
    }
无限循环:while(true){} 或for(;;){}
    无限循环存在的原因是并不知道循环多少次，而是根据某些条件，来控制循环
循环嵌套:在一个循环语句的循环体中再定义一个循环语句的语法结构

    跳转语句

    break语句
在switch条件语句中时，终止某个case并跳出switch结构
在循环语句中，作跳出本层循环语句，执行后面的代码
标记：当break语句出现在嵌套循环中的内层循环时，它只能跳出内层循环，
如果想使用break语句跳出外层循环则需要对外层循环添加   标记XXX：然后break XXX

public class BreakDemo {
    public static void main(String[] args) {
        int i, j; // 定义两个循环变量
        itcast: for (i = 1; i <= 9; i++) { // 外层循环
            for (j = 1; j <= i; j++) { // 内层循环
                if (i > 4) { // 判断i的值是否大于4
                    break itcast; // 跳出外层循环
                }
                System.out.print("*"); // 打印*
            }
            System.out.print("\n"); // 换行
        }
    }
}//只打印了4个*

continue语句：终止本次循环，执行下一次循环
continue语句后面也可以通过使用标记的方式结束本次外层循环，用法与break语句相似

    数组（数组长度是固定的，数组中存储的元素的数据类型要求一致）

    分类 说明 代码
    一维数组 定义 动态初始化（定义数组时只指定数组长度，由系统自动为元素赋初值）
    数据类型[] 数组名 = new 数据类型[数组长度];
    静态初始化（定义数组时为元素赋值）
    类型[] 数组名 = new 类型[]{元素，元素，……};
    类型[] 数组名 = {元素，元素，元素，……};(建议)
    访问 数组名[索引]、数组名[索引] =值、数组名.length
    遍历 for (int i = 0; i < arr.length; i++) {System.out.println(arr[i]);}
    最值 int max=arr[0];for(int x=1;x<arr.length;x++){if(arr[x] > max){max = arr[x];}}
    二维数组 定义 int[][] arr = new int[m][n]
    m:二维数组中一维数组的个数,n:每个一维数组中元素的个数
    int[][] arr = new int[3][];
    m:代表二维数组中一维数组的个数,每一个一维数组通过赋值来确定数组长度
    int[][] arr = {{1,2},{3,4,5,6},{7,8,9}}
