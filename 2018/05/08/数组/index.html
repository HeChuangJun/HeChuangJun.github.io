数组

May 8, 2018 | JavaSE

数组是用来存储同一个数据类型多个元素的一个容器
（数组长度是固定的，数组中存储的元素的数据类型要求一致）

一维数组

数组定义：
    ①动态初始化（定义数组时只指定数组长度）
        数据类型[] 数组名 = new 数据类型[元素个数或数组长度];
        int[] x;// 声明一个int[]类型的变量
        x = new int[100];// 创建一个长度为100的数组
        int[] x; 声明了一个变量x，该变量的类型为int[]，即一个int类型的数组。
        变量x会占用一块内存单元，它没有被分配初始值。
        x = new int[100]; 创建了一个数组，将数组的地址赋值给变量x。
        在程序运行期间可以使用变量x来引用数组，这时内存中的状态会发生变化
    ②静态初始化（定义数组时为元素赋值）
        类型[] 数组名 = new 类型[]{元素，元素，……};
        类型[] 数组名 = {元素，元素，元素，……};（建议）

    访问
        数组名[索引/角标]   索引从0开始到数组长度-1
        数组名[索引/角标] =值
        数组长度：数组名.length

    遍历
        for (int i = 0; i < arr.length; i++) {
            System.out.println(arr[i]); // 通过索引访问元素
        }

    数组最值
        int[] arr = { 4, 1, 6, 3, 9, 8 }; 
        int max = arr[0];
        for (int x = 1; x < arr.length; x++){
            if (arr[x] > max) { 
                max = arr[x];
            }
        }

二维数组

是一个包含多个一维数组的数组，数组中每个元素都是一维数组
定义
    int[][] arr = new int[m][n];
        m: 代表二维数组中一维数组的个数、n: 代表每个一维数组中元素的个数
    int[][] arr = new int[3][];
        m: 代表二维数组中一维数组的个数\每一个一维数组通过赋值来确定数组长度
    int[][] arr = {{1,2},{3,4,5,6},{7,8,9}};

访问
    数组名[索引][索引]
    数组名[索引][索引]=值
    数组名.length

遍历
    for (int i=0; i<arr2.length; i++) {
        for (int j=0; j<arr2[i].length; j++) {
            sum2 += arr2[i][j];
        }
    }

数组异常

空指针异常NullPointerException：在使用变量引用一个数组时，变量必须指向一个有效的数组对象，
如果该变量的值为null，则意味着没有指向任何数组，此时通过该变量访问数组的元素会出现空指针异常，
数组越界异常ArrayIndexOutOfBoundsException
