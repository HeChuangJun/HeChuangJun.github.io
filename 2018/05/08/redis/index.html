    Redis集群的搭建

    Redis集群中至少应该有三个节点。要保证集群的高可用，需要每个节点有一个备份机。 Redis集群至少需要6台服务器。 搭建伪分布式。可以使用一台虚拟机运行6个redis实例。需要修改redis的端口号7001-7006

    集群搭建环境

    1、使用ruby脚本搭建集群。需要ruby的运行环境。 安装ruby yum install ruby yum install rubygems

    安装ruby脚本运行使用的包。

    将redis-3.0.0和redis-3.0.0.gem放在一起然后执行 gem install redis-3.0.0.gem
