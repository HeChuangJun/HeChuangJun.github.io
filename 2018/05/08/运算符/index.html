运算符

May 8, 2018 | JavaSE

运算符:用来计算数据的符号。数据可以是常量，也可以是变量。被运算符操作的数我们称为操作数

算术运算符:+、+、+、-、-、*、/、%、++、–

将操作数参与数学计算
    加法运算符在连接字符串时只有直接与字符串相加才会转成字符串
    “%”小数取余没有意义。结果符号与被取余符号相同
    “/”当两边为整数时，取整数部分，舍余数
    当其中一边为浮点型时，按正常规则相除。整数做被除数，0不能做除数
    ++,--运算符单独使用，不参与运算操作时，运算符前后位置导致的运算结果一致
    ++,--运算符后置时，变量a先参与运算操作，然后变量a自增1或者自减1
    ++，--运算符前置时，变量a的值先自增1或者自减1，然后参与运算操作

赋值运算符:=、+=、-=、*=、/=、%=

赋值运算符，会将结果自动强转成等号左边的数据类型
    赋值运算符左边必须是变量

比较运算符:==、！=、<、>、<=、>=

返回布尔值，判断两个操作数的大小关系及是否相等关系
    赋值运算符的=符号，是用来将=符号右边的值，赋值给=符号左边的变量
    比较运算符的 == 符号，对于基本数字类型的变量来说是用来判断 == 符号 
    左右变量的值是否相等的，而对于对象来说，比较的是内存地址

逻辑运算符:&（与）、|、^、！、&&、||

连接两个其他表达式计算后的布尔值结果
    当使用短路与或者短路或时，只要能判断出结果则后边的部分就不再判断。
    异或^： 左右两边条件结果相同，结果就为false，左右两边条件结果不同，结果就为true；

三元运算符:(条件表达式)？表达式1：表达式2；

先判断条件表达式的值，若为true，运算结果为表达式1；若为false，运算结果为表达式2
    System.out.println( 3>2 ? “正确” : “错误” ); 
    int a = 3;int b = 4;String result = (a==b) ? “相等” : “不相等”;  
    int n = (3>2 && 4>6) ? 100 : 200;
