ajax异步的JavaScript和XML

ajax(Asynchronous JavaScript And XML)

分类 现象
同步 客户端发送请求到服务器端，当服务器返回响应之前，客户端都处于等待卡死状态
异步 客户端发送请求到服务器端，无论服务器是否返回响应，客户端都可以随意做其他事情，不会被卡死

js原生ajax-通过本地浏览器中的ajax引擎（只能）

分类 现象
创建Ajax引擎对象(谷歌,火狐,IE) if(window.XMLHttpRequest){var xmlhttp= new XMLHttpRequest();}
创建Ajax引擎对象(其他) if(window.ActiveXObject){var xmlhttp=new ActiveXObject(Microsoft.XMLHTTP);}
为Ajax引擎对象绑定监听 xmlhttp.onreadystatechange=function(){
（监听服务器已将数据响应给引擎） if(xmlhttp.readyState==4&&xmlhttp.status==200){alert(xmlhttp.responseText);}}
get提交 xmlhttp.open(“GET”,”$(pageContext.request.contextPath)/Ajax?xxx=xx&xx=xx”,true)
post提交 xmlhttp.open(“POST”,”$(pageContext.request.contextPath)/Ajax”,true)
post提交在发送前设置请求头 xmlhttp.setRequestHeader(“Content-type”,”application/x-www-form-urlencoded”);
发送请求 xmlhttp.send(null);//get
xmlhttp.send(xxx=xx&xx=xx)//post
服务器获得ajax的参数 request.setCharacterEncoding(“UTF-8”);post提交方式
String name = request.getParameter(“name”);
服务器返回ajax字符串 response.setContentType(“text/html;charset=UTF-8”);
response.getWriter().write(“{\”name\”:\”汤姆\”,\”age\”:21}”);
