ajax异步的JavaScript和XML

ajax（重点）

同步&异步

同步现象：客户端发送请求到服务器端，当服务器返回响应之前，
    客户端都处于等待卡死状态
    异步现象：客户端发送请求到服务器端，无论服务器是否返回响应，
    客户端都可以随意做其他事情，不会被卡死

运行原理

页面发起请求，会将请求发送给浏览器内核中的Ajax引擎，
    Ajax引擎会提交请求到服务器端，在这段时间里，客户端可以任意进行任意操作，
    直到服务器端将数据返回给Ajax引擎后，会触发你设置的事件，
    从而执行自定义的js逻辑代码完成某种页面功能。
    所用异步访问都是ajax引擎  AJAX即Asynchronous JavaScript And XML
    不需要刷新网页就能对网页某部分进行更新，且不影响本来页面的加载

js原生ajax-通过本地浏览器中的ajax引擎(了解)

1）创建Ajax引擎对象
    var xmlhttp= new XML HttpRequest();
    2）为Ajax引擎对象绑定监听（监听服务器已将数据响应给引擎）
    xmlhttp.onreadystatechange=function(){
    if(xmlhttp.readyState==4&&xmlhttp.status==200){
    var res=xmlhttp.responseText;
    alert(res);
    }
    }
    3）绑定提交地址
    xmlhttp.open("GET","$(pageContext.request.contextPath)/Ajax",true)
    4）发送请求
    xmlhttp.send();
    如果是post提交，在发送请求之前设置一个头
    xmlhttp.setRequestHeader
    ("Content-type","application/x-www-form-urlencoded");
    只能谷歌或火狐浏览器使用
    XMLHttp对象浏览器兼容

jquery的ajax

$.post(url,data,function,type);
    $.get(url,data,function,type);
    $.ajax({})
    $.ajax({
    url:"/WEB22/ajaxServlet2",
    async:true,//是否异步，默认是true代表异步
    type:"POST",//请求方式，POST/GET 
    data:{"name":"lucy","age":18},
    //发给servlet的请求对象(发到服务器的参数)建议json格式
    success:function(data){//成功响应执行的函数，对应的类型是function类型
    alert(data.name);//data,从servlet发过来的数据对象
    },
    error:function(){
    alert("请求失败");
    },
    dataType:"json"//服务器端返回的数据类型，常用text和json
    });

参数

url：代表请求的服务器端地址
    data：代表请求服务器端的数据（可以是key=value形式也可以是json格式）
    callback：表示服务器端成功响应所触发的函数（只有正常成功返回才执行）
    一般都是写funtion（data）{},其中data就是相应回来的json格式对象
    type：表示服务器端返回的数据类型（jquery会根据指定的类型自动类型转换）
    常用的返回类型：text、json、html等

json格式

对象格式：{“key1”:obj,”key2”:obj,”key3”:obj…}

json的key是字符串  jaon的value是Object，没有双引号表示变量
    user对象 json数据格式{"username":"zhangsan","age":28,"password":"123"}
    List<Product> 用json数据格式：[{"pid":"10","pname":"小米4C"},{},{}]

数组/集合格式：[obj,obj,obj…]

对象格式和数组格式可以互相嵌套
    json的解析：json是js的原生内容，也就意味着js可以直接取出json对象中的数据
    语言无关的数据交换的格式，轻量级的数据交换格式
    可以使用ajax进行前后台数据交换，移动端与服务端的数据交换
