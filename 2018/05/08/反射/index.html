类加载器
分类
Bootstrap ClassLoader 根类加载器 /引导类加载器，
负责Java核心类的加载 比如System,String等。在JDK中JRE的lib目录下rt.jar文件中
Extension ClassLoader 扩展类加载器
负责JRE的扩展目录中jar包的加载。在JDK中JRE的lib目录下ext目录
System ClassLoader 系统类加载器
负责在JVM启动时加载来自java命令的class文件，以及classpath环境变量所指定的jar包和类路径
功能
负责将.class文件加载到内存中，并为之生成对应的Class对象
反射使用
反射机制概念
JAVA反射机制是在运行状态中，对于任意一个类，都能够知道这个类的所有属性和方法；对于任意一个对象，都能够调用它的任意一个方法和属性；这种动态获取的信息以及动态调用对象的方法的功能称为java语言的反射机制。
自己的理解
JAVA的反射机制就是利用类加载进入内存后（产生.class字节码文件、该类的对象、还有.class字节码文件对应的class对象三个东西），通过获得类加载器自动class字节码文件封装成对应的class对象并利用class对象的方法获得class文件所有的属性和方法，然后就能执行创建对象，调用构造方法，一般方法，设置属性值等操作，不用修改源码就能解决动态调用方法的问题
①获得class类对象对字节码文件的描述对象
（注意有且只有一个）
不能创建，只能获得
API
Class 没有公共构造方法。Class 对象是在加载类时由 Java 虚拟机以及通过调用类加载器中的 defineClass 方法自动构造的
获得方式
通过Object类中的getObject()方法,必须明确Person类型.
Person p = new Person(); Class c = p.getClass();
通过 类名.class 获取到字节码文件对象（任意数据类型都具备一个class静态属性）
Class c2 = Person.class; 必须明确Person类型.
通过Class类中的方法（将完整类名作为字符串传递给Class类中的静态方法forName即可）。
Class c3 = Class.forName(“cn.junye.Person”);
指定这种类型的字符串就行.这种扩展更强.不需要知道类.我只提供字符串,按照配置文件加载就可以了
②class类获得构造方法的方法及其返回封装好的Constructor类
返回一个构造方法
public Constructor getConstructor(Class<?>… parameterTypes) 获取public修饰, 指定参数类型所对应的构造方法Xxx.class/null
public Constructor getDeclaredConstructor(Class<?>… parameterTypes) 获取指定参数类型所对应的构造方法(包含私有的)
返回多个构造方法
public Constructor<?>[] getConstructors() 获取所有的public 修饰的构造方法
public Constructor<?>[] getDeclaredConstructors() 获取所有的构造方法(包含私有的)
③利用Constructor类创建对象的方法
public T newInstance(Object… initargs)//传递实际参数/null
②class类获得成员变量的方法及其返回封装好的Field类
返回一个成员变量
public Field getField(String name) 获取指定的 public修饰的变量
public Field getDeclaredField(String name) 获取指定的任意变量
返回多个成员变量
public Field[] getFields() 获取所有public 修饰的变量
public Field[] getDeclaredFields() 获取所有的 变量 (包含私有)
③利用Field类对属性进行赋值取值的方法
（必须创建对象以传入方法）
public void set(Object obj, Object value)在指定对象obj中，将此 Field 对象表示的成员变量设置为指定的新值
public Object get(Object obj)返回指定对象obj中，此 Field 对象表示的成员变量的值
②class类获得成员方法的方法及其返回封装好的Method类
返回获取一个方法：
public Method getMethod(String name, Class<?>… parameterTypes)
获取public 修饰的方法
public Method getDeclaredMethod(String name, Class<?>… parameterTypes)
获取任意的方法，包含私有的
参数1: name 要查找的方法名称； 参数2： parameterTypes 该方法的参数类型
返回获取多个方法：
public Method[] getMethods() 获取本类与父类中所有public 修饰的方法
public Method[] getDeclaredMethods() 获取本类中所有的方法(包含私有的)
③利用Method类对象执行class文件中的成员方法相应的方法
（必须创建对象以传入方法）
public Object invoke(Object obj, Object… args)
执行指定对象obj中，当前Method对象所代表的方法，方法要传入的参数通过args指定
AccessibleObject 是Field类、Method类、Constructor类的父类，
访问class内容时自动调用访问控制语言检查的方法，访问私有修饰的东西必须修改，在获得以上三个类后调用，即可取消
暴力访问私有构造方法、一般方法、成员变量，在访问前设置，true为取消Java语言访问检查
public void setAccessible(boolean flag) throws SecurityException
