div+css

div+css3

div概念

div是html块级标签，进行区域划分，独自占一行，后来加上的东西只能拎起一行，
    单独使用无复杂效果，必须结合css样式进行渲染，用于页面布局
    span是html标签行内元素，默认所有的在一行，span后面加的东西可以和它同一行，
    都可以进行区域划分，单独使用无意义，必须结合CSS使用，
    用于对括起来的内容进行样式的修饰，一般用于字体修饰

CSS（cascacingstylesheet)样式或层叠样式表

它是用于控制网页样式并允许将样式信息与网页内容分离的一种标记性语言
    用于设置HTML页面中的文本内容（字体、大小、对齐方式等）、
    图片的外形（高度、边框样式、边距等）
    以及版面的布局等外观显示样式，就是使html页面更好看。
    样式：给HTML标签添加需要显示的效果
    样式通常存储在样式表中，层叠：使用不同的添加方式
    給同一个HTML标签添加样式，最后所有的样式叠在一起共同作用于该标签

语法

选择器{属性1：属性值；属性2：属性值；..}
    选择器区分大小写，属性和属性值不区分大小写，
    如果属性值中间由多个单词组成且中间包含空格，
    则必须为这个属性值加上英文双引号

选择器:CSS寻找实现效果的目标元素的工具（核心）

基本选择器
id选择器：#id
    类选择器：.class
    元素选择器：html标签名
    对该类型的所有标签都有效，能快速对页面同类型标记统一样式
    无法设计差异化样式（对相同内容设置不同样式时出现标签不够用的情况）
    用于对多个元素设置相同的样式
    层级（包含）选择器
        父元素    子元素  如：div p
    属性选择器
        基本选择器[属性=‘属性值’]   如：input[type='text']
    注意
    一个类名可作用于多个标签上面，一个标签也可有多个类名(多个类名用空格隔开)
    一个 id 只可以作用于一个标签上面，一个标签也只有一个 id 名。
    类选择器与id选择器在命名只能是英文字母、数字、连接符(-)、下划线(_)，
    数字不能打头。(连接符、下划线打头后面需要跟字母)
属性：属性值
border边界必设置
border: 1px solid black;  solid实线、none无边、double双线
width必设置
width:10px;
height必设置
height:10px;
float
float:left/right/none;//浮动
    clear:left/right/both//清除浮动 浮动不再占用原文档流的位置
display
block块标签，以区域的方式出现，每个标签独自占据一整行或多整行。a,span
    inline行内元素，不必再新的一行开始，不强迫其他元素在新一行显示hn,div,ul,ol
    inline-block将对象呈递为内联对象，但是对象的内容作为块对象呈递
    none,不显示不占用页面空间
margin外边距
margin: auto;页面居中
padding内边距
与font相关的属性
font-style
    font-style：italic(斜体)、normal(正常的);
    font-size（必写、最后）font-size: 30px;
    font-family（必写、最后）font-family:"楷体";
    color用于设置div里面字体的颜色
    没有font-color属性！！！！
    (1)(英文单词)red;(2)十六进制数字， color：#FF00FF
    (3)rgb 表示rgb(0,0,0)，三个值红、绿、蓝，每个取值都是 0-255 或者百分比
    font-weight 
    数字(100-900 的整百数值)、bold加粗，bolder 比加粗更粗，lighter 细线)
    text-align: center;内容（包括图片文字）居中
    text-decoration: none;超链接的下划线去掉
背景属性
background-color：red；background-image

引入CSS样式

行内引入

<div style="font-size: 20px;color: blue;">
    利用标签的style属性来控制样式，
    没有做到结构与表现（HTML展示结构、CSS显示效果）分离，耦合性大，少用

内部引入（内嵌式）

<style type="text/css（可省略。默认）">
    选择器{
    属性：属性值;
    }
    </style>//在head标签之间
    只对其所在的HTML页面有效，可以对多处标签统一设置样式，
    设计单页面使用，设计网站不用，因为无法发挥代码重用性

外部引入（链入式）

在head之间<linkrel="stylesheet（指定为样式的脚本）
    "href="/？.css（指定css的路径）"type="text/css"/>
    将多由的样式放在一个或多个以.css为扩展名的外部文件中，
    再通过link标签将样式连接到HTML文档中，解决内容与表现分离，
    同一个样式表可以被不同页面连接使用（方便后期维护），
    同时一个页面可以通过多个<link/>标记链接多个CSS样式表

优先级

就近原则 理论上：行内>内嵌>链接
    内嵌、链接、导入在同一个文件头部，谁离相应的代码近，谁的优先级高

盒子模型

Box Model规定类元素框处理元素内容、内边距、边框和外边距的方式

内边距padding

h1{padding:10px;}
    h1{padding:10px 0.25em 2ex 20%;}按上右下左的顺序设置各边的内边距，
    可为像素百分比等不同单位
    padding-top、padding-right、padding-bottom、padding-left

边框：border

border-top-style、border-right-style、
    border-bottom-style、border-left-style

外边距：margin

margin-top、margin-right、margin-bottom、margin-left

取值问题

在 CSS 中，width 和 height
    指的是内容区域的宽度和高度。
    增加内边距、边框和外边距不会影响内容区域的尺寸，
    但是会增加元素框的总尺寸。
    而默认情况下，内边距、边框和外边距的值均为 0.
    设置了之后要使原来的总尺寸不变，
    必须重新设置width和height

定位问题

定义元素框相对于其正常位置应该出现的位置，
    或者相对于父元素、另一个元素甚至浏览器窗口本身的位置
    机制：普通流（默认）、浮动、绝对定位
普通流（默认）
相对定位：文档流，占据空间（可能覆盖），
    使该元素框相对地出现在本该出现的普通流的位置多少像素之外。
    利用position：relative控制，同时设置top、left、right、bottom
    #box_relative {
    position: relative;
    left: 30px;
    top: 20px;
    }
绝对定位：与文档流无关，不占据空间
#box_relative {
    position: absolute;
    left: 30px;
    top: 20px;
    }

浮动

浮动的框可以向左或向右移动，直到外边缘碰到包含框或另一个浮动框的边框为止
    由于浮动框不在文档的普通流中，文档的普通流中的块框就像浮动框不存在一样
浮动的框可能覆盖不浮动（文档流）框，
浮动框碰到浮动框按顺序排列，
如果包含框太窄，无法容纳水平排列的三个浮动元素，那么其它浮动块向下移动，直到有足够的空间。
如果浮动元素的高度不同，那么当它们向下移动时可能被其它浮动元素“卡住”：
清除浮动

使用div+css的步骤构建页面的步骤（个人看法）

1、先分析网页由多少个div组成
    2、根据具体div设置相应选择器及样式width、height、border必写，
    根据F12的开发人员工具来完成以上问题的设置
    3、最后慢慢调整间隙，注意浮动的妙用。
