xml约束

DTD

概念
    文档类型定义，约束XML元素及其子元素的名称及顺序、属性等
文档声明
内部DTD:<!DOCTYPE 根标签名  [语法....]>只对当前的xml有效
    外部DTD--本地文件:在<?xml version="1.0" encoding="UTF-8"?>下面
    <!DOCTYPE web-app SYSTEM "web-app_2_3.dtd">
    格式<!DOCTYPE 根标签名 SYSTEM "xxxx.dtd">
    外部DTD--网络:<!DOCTYPE 根标签名 PUBLIC "名称" "路径">一般由框架提供
元素声明<!ELEMENT 标签名 描述>
描述组成：符号，约束
    符号：+ * ？  , | ()   “文嘉兴”
    约束：#PCDATA 表示内容是文本不是子标签
    后面没加符号的表示有且只有一次，()分组并指示子元素个数及顺序
属性声明<!ATTLIST 标签名 属性描述 属性描述2 …>
属性描述：属性名  属性类型  属性约束
    名称：自定义
    类型：CDATA ，ID ，枚举（xx|）
    约束：#REQUIRED （必须） #IMPLIED（可选）

Schema

概念
    xml文档约束，比DTD功能强、扩展名为xsd、支持命名空间、数据类型更完善
命名空间
文档声明
<xxx xmlns="...."xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
     xsi:schemaLocation="名称  路径   名称2 路径2  .....">  .....
    <web-app xmlns="http://www.example.org/web-app_2_5" 自定义文档命名空间（包名）
    xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"（官方文档命名空间）
    xsi:schemaLocation="http://www.example.org/web-app_2_5 web-app_2_5.xsd" 
    使用了官方文档的自定义文档位置属性（包里面的具体类）version="2.5">
    命名空间namespace，就是给schema文档起一个名称，只是建议采用是url进行命名。
    是不是官方文档看那个url

目的：在xml中区别多个约束文档中，重名的元素、属性等内容。

缺省命名空间（默认）使用元素或属性时不加前缀

显示命名空间 使用元素或属性时加前缀<别名:xxx>

只要url一样，两种声明方式一样

xml解析

xpath

解析方式

DOM解析

将xml文档加载到内存形成树结构，形成Document对象
    优点:元素与元素有结构关系，能CRUD，
    缺点：文档太大易内存溢出

SAX

事件驱动的方式逐行边读边解析，每执行一行都触发相应的事件
    优点：文档大也不内存溢出，速度快
    缺点：只读，不能CRUD，运行后释放资源

PULL:解析开发包

DOM解析原理及结构模型
    原理:将整个xml文档加载到内存，生成一个DOM树并获得Document对象，通过该对象完成DOM操作    
    API使用
    ①引入jar包dom4j-1.6.1.jar
    ②获得saxreader对象
    SAXReader saxReader = new SAXReader();
    ③获得Document
    Document doc = saxReader.read("src/cn/itheima/xml/schema/web.xml");
    ④获得根元素
    Element rootElement = doc.getRootElement();
    ⑤获得元素
    List elements()获得指定名称的所有子元素，可以不指定名称
    element element("servlet-name")获得指定名称的第一个子元素
    遍历所有元素
        for (Element element : childElements) {
        if ("servlet".equals(element.getName())) {
        Element servletName = element.element("servlet-name");
        Element servletClass = element.element("servlet-class");
        }
        }
    ⑥获得元素内容
    attributeValue("XXX")根据属性名得到属性值
    getName()获得当前元素的元素名
    getText()获得当前元素的文本内容
    string elementText（“xxx”）=element（xxx）.getText()
