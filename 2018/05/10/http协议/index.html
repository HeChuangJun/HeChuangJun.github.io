http协议

http协议

概念：超文本传输协议，用于定义WEB服务器和WEB浏览器之间交换数据的过程

特点：HTTP协议默认端口80，响应和请求必须成对，先有请求后有响应

版本

HTTP/1.0 发送请求，创建一次连接，获得一个web资源，连接断开
    HTTP/1.1 发送请求，创建一次连接，获得多个web资源，连接断开(只用这个)

组成

Http请求协议

请求行：POST/day09/01.http/from.html HTTP/1.1请求格式的第一行
请求方式 ：get：请求参数在url后面，长度限制，没有请求体
    Post：请求参数在请求体，长度无限制，安全
    除了表单的method为post外，其他都是get请求                        
    资源路径 url
    统一资源定位符是对互联网上资源位置的一种表示，每个文件都有唯一的URl
