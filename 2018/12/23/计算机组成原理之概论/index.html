计算机组成原理之概论

Dec 23, 2018 | 计算机

计算机系统概论

计算机系统简介
    计算机的软硬件概念
        计算机系统由"硬件"和"软件"两大部分组成
        硬件：计算机实体部分，由各种电子元器件，各类光、电、机设备的实物组成
        软件：由人们事先编制的具有各类特殊功能的程序组成，常存放在计算机的主存或辅存内，分为系统软件和应用软件
            系统软件(系统程序)：管理整个计算机系统，监视服务，使资源得到合理调度，高效运行，包括标准程序库、语言处理程序、操作系统、数据库管理系统、网络软件等
            应用软件(应用程序)：根据任务需要编制的各种程序，如科学计算程序，数据处理程序，过程控制程序，事务管理程序等
    计算机系统的层次结构

    计算机组成和计算机体系结构
        计算机体系结构：指那些能够被程序员所见到的计算机系统的属性，即概念性的结构与功能
            包括指令集、数据类型、存储器寻址技术、i/o机理等大多数是抽象属性
        计算机组成：指如何实现计算机体系结构所体现的属性
            指令系统属于计算机结构问题，而指令的实现、如何取指令、分析指令、取操作数、运算、送结果等属于计算机组成问题
        结构相同的计算机，组成（结构实现）不一定相同
计算机基本组成
    冯诺依曼计算机的特点
        1945年，数学家冯诺依曼提出存储程序的概念，以此概念为基础的计算机都称为冯诺依曼机
        计算机是由运算器、存储器、控制器、输入设备和输出设备五大部件组成
        指令和数据均以二进制表示，存放在存储器内，并可按地址寻访
        指令由操作码和地址码组成，操作码用来表示操作的性质，地址码用来表示操作数在存储器中的位置
        指令在存储器内按顺序存放。通常指令是顺序执行的，在特性条件下，可根据运算结果或根据设定条件改变执行顺序
        机器以运算器为中心，输入输出设备与存储器间的数据传送通过运算器完成
    计算机的硬件框图

        各部件功能
            运算器：完成算术运算和逻辑运算，并将运算结果暂存在运算器内
            存储器：存放数据和程序
            控制器：控制、指挥程序和数据的输入、运行以及处理运算结果
            输入设备：将人们熟悉的信息形式转换为机器能识别的信息形式
            输出设备：将机器运算结果转换为人么熟悉的信息形式
        相关概念
            中央处理器CPU(Central Processing Unit) = 运算器（算术逻辑单元ALU） + 控制器（控制单元CU）
            I/O设备(外部设备) = 输入设备 + 输出设备
            现代计算机 = CPU + I/O设备 + 主存储器
            主机 = CPU + 主存储器
    计算机的工作步骤（待补充）
计算机硬件的主要技术指标
    机器字长
        CPU一次能处理数据的位数，与CPU寄存器位数有关，字长越长，数的表示范围越大，精度越高
    存储容量
        包括了主存容量和辅存容量，主存容量指主存中存放二进制代码的总位数
        存储容量 = 存储单元个数 x 存储字长
        存储单元个数 = 2^MAR的位数
        MDR的位数 = 存储字长
    运算速度
        MIPS：单位时间内执行指令的平均条数（百万条指令每秒）
        CPI：执行一条指令所需的时钟周期（主频的倒数）
        FLOPS：浮点数运算次数每秒

计算机发展简史

计算机发展的四个阶段
1946-1957：电子管计算机（第二次世界大战的ENIAC计算机）集成度小，空间占用大，功耗高，运算速度慢、操作复杂，更换程序需要接线
1957-1964：晶体管计算机（贝尔实验室发明）集成度相对较高，空间占用相对小，功耗相对较低，运行速度较快，操作相对简单。交互更加方便
1964-1980：集成电路计算机（德州仪器发明了集成电路IC）计算机变得更小，功耗变得更低，计算速度变得更快
1980-现在：超大规模集成电路计算机 一个芯片集成了上百万的晶体管，速度更快，体积更小，价格更低，更能被大众接受
用途丰富：文本处理、表格处理、高交互的游戏与应用

微型计算机的发展历史
受限于性能，单核CPU-》多核CPU
摩尔定律：当价格不变时，集成电路上可容纳的器件的数目，每18-24个月会提升一倍，性能也将提升一倍

计算机的分类

超级计算机：功能最强、运算速度最快、存储容量最大的计算机，用于科研
（1TFlop/s=每秒一万亿次浮点计算）
大型计算机：大型机、大型主机、主机等，高性能、可处理数据与复杂的运算
迷你计算机（服务器）：小型机、普通服务器、不需要特殊的空调场所，具备不错的运算能力，可以完成复杂的运算
工作站：高端的通用微型计算机，提供比个人计算机更强大的性能，类似于普通台式电脑，体积较大，但性能强劲
微型计算机：个人计算机 台式机 笔记本电脑 一体机 麻雀虽小，五脏俱全

计算机的体系结构

冯诺依曼体系（以运算器为核心的结构）：
    将程序指令和数据一起存储的计算机设计概念结构（存储程序指令，设计通用电路）
基本组成：
    存储器、控制器、运算器、输入设备、输出设备
功能：
    能够把需要的程序和数据送至计算机中
    能够长期记忆程序、数据、中间结果及最终运算结果的能力
    能够具备算术、逻辑运算和数据传送等数据加工处理的能力
    能够那招要求将处理结果输出给用户
工作原理：
    程序指令和运行数据存储起来，使得计算机从专用电路计算机发展成为通用电路计算机。
冯诺依曼瓶颈（将存储器和运算器分开，以运算器为核心）：
    运算器运算速度比存储器访问速度快（CPU经常空转等待数据传输）
现代计算机的结构（以存储器为核心的结构）
    根据局部性原理设计了存储器的层次结构，大幅提升计算机性能。
    解决CPU与存储设备之间的性能差异问题
    其中存储器主要指相对高速的设备（内存、CPU的寄存器）

计算机的层次和编程语言

程序翻译和程序解释
    计算机执行的指令都是L0
    翻译过程生成新的L0程序
    解释过程不生成新的L0程序
    解释过程由L0编写的解释器去解释L1程序
    程序解释指的是程序代码在运行时，逐行翻译成较低层次的编程语言去执行；
    程序翻译指的是程序代码在编译阶段提前翻译成较低层次的编程语言逻辑，在运行时直接运行
程序翻译（C\C++\Golang）
程序解释（Python\Php\Javascript）
翻译+解释（java、C#）

计算机层次和编程语言

微指令<微程序=机器指令  ：一条机器指令对应一个微程序，一个微程序对应一组微指令
硬件逻辑层：门、触发器等逻辑电路组成，属于电子工程的领域
微程序机器层：编程语言是微指令集，微指令所组成的微程序直接交由硬件执行
传统及机器层：编程语言是CPU指令集（机器指令），编程语言和硬件直接相关，不同架构的CPU使用不同的CPU指令集
操作系统层：向上提供了简易的操作界面，向下对接了指令系统，管理硬件资源，是软件和硬件之间的适配层
汇编语言层：编程语言是汇编语言，汇编语言可以翻译成直接执行的机器语言。完成翻译的过程的程序就是汇编器
高级语言层：广大程序员接受的高级语言，类别多，几百种，常见Java等
应用层:word excel ppt

高级语言、汇编语言和机器语言的区别和联系

高级语言是方便程序员描述程序逻辑的编程语言，与人类语言接近；
汇编语言时方便程序员描述硬件逻辑的编程语言，与底层硬件接近；
机器语言时方便计算机理解和运行的编程语言，一般程序员无法理解

计算机的计算单位

容量单位
    在物理层面，高低电平记录信息，理论上只认识0/1（比特位）两种状态
    1字节（byte）= 8比特位（bits） 1024 = 2^10
    硬件一般用1000进制表示，与1024进制有所区别，所以500G硬盘实际比500G少

速度单位
    网络速度(单位为bps)：
        bps，比特率是指每秒传送的比特(bit)数。单位为 bps(Bit Per Second)，比特率越高，每秒传送数据就越多
        电信100M光纤，指的是100M/s，峰值速度只有12M/s? 100M/s = 100 Mbps = 100Mbit/s = (100/8)MB/s = 12.5MB/s
    CPU速度：CPU的时钟频率
        单位赫兹（Hz）：秒分之一
        每秒中的周期性变动重复次数的计量
        高低电平每秒钟变动的频率
        1Hz = 1/s，即在单位时间内完成振动的次数，单位为赫兹（1赫兹=1次/秒）。
        2GHz=2*1000^3Hz=每秒20亿次（变动高低电平）

计算机的字符和编码集

字符编码集的历史
    ASCII码
        7bits完全表示ASCII码
        95个可打印字符、33个不可打印的字符（包括控制字符）
        总共128个字符（2^7）
        但很多应用或者国家中的符号无法表示（数学符号、中文）

Extended ASCII码
    7bits->8bits
    常见的数学运算符、带音标的欧洲字符、其他常用符、表格符等

    字符编码集的国际化
        语言国际化（不以有限字符组合的语言）
中文编码集
    中国国内用
        GB2312（信息交换用汉子编码字符集---基本集） 收录了7445个字符（6763个汉字和682个其他字符）
        GBK（汉字内码扩展规范）、向下兼容GB2312、向上支持国际ISO标准，收录了21003个汉字，支持劝募中日韩汉字
    Unicode 兼容全球的字符集
        Unicode：统一码、万国码、单一码
        Unicode定义了世界通用的符号及，UTF-*实现了编码
        UTF-8以字节为单位对Unicode进行编码
        编程推荐使用UTF-8、windows默认GBK编码
