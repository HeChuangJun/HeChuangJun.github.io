art-template

Jul 30, 2018 | art-template

模板引擎有很多，先学习一个
注意只要有 匹配，不管你在字符串里面还是在哪，都会与template关联

介绍

art-template 是一个简约、超快的模板引擎。
    它采用作用域预声明的技术来优化模板渲染速度，
    从而获得接近 JavaScript 极限的运行性能，
    并且同时支持 NodeJS 和浏览器。在线速度测试。

art-template用于渲染html

可用于服务端
服务端渲染能被搜索引擎搜到，用户体验没那么好
客户端渲染不能被搜索引擎搜索到，但是提高了用户体验

art-template在浏览器中的使用

art-template的安装

在某个地方使用cmd执行这个命令即可，默认下载到执行命令的文件夹下
    npm install art-template --save

导入js环境

使用核心方法

// 基于模板名渲染模板(浏览器使用)
template(filename, data);

// 将模板源代码编译成函数
template.compile(source, options);

// 将模板源代码编译成函数并立刻执行
template.render(source, data, options);

在浏览器使用例子(替换模板)，以下的遍历只能于模板引擎

<script src="node_modules/art-template/lib/template-web.js"></script>
    <script type="text/template" id="test">
        {{ name }},{{ age }},{{each hobbies}} {{$value}} {{/each}}    
    </script>
    <script>
        var ret=template('test',{
            name:'jack',
            age:18,
            hobbies:[
                '写代码',
                '玩游戏',
                '唱歌'
            ]
        })
        console.log(ret);
    </script>

在node中使用（字符串替换）

var template =require('art-template');
    var str='{{ name }},{{ age }},{{each hobbies}} {{$value}} {{/each}}    ';
    var htmlstr=template.render(str,{
            name:'jack',
            age:18,
            hobbies:[
                '写代码',
                '玩游戏',
                '唱歌'
            ]
        });
    response.end(htmlstr);
